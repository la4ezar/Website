HTML5 Markup

For Web coding I use HTML5. It is the latest evolution of the standard that defines HTML. It is a markup language used for structuring and presenting content on the World Wide Web.

Framework

I'm using only Bootstrap for now. It contains HTML and CSS based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions.

CSS3 Styling

For styling my elements I use CSS3. It is the latest standard for CSS and the primary language used to describe look and formatting for webpages across the Internet and documents of markup in general.
